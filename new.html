where the GuzzleHttp\Psr7\Request constructor’s parameters are:

Parameter Meaning types allowed
$method $uri $headers $body $version HTTP method–GET, POST, DELETE, etc route headers request body Protocol version (defaults) string string|UriInterface array<string, string|string[]> string|resource|StreamInterface|null string
